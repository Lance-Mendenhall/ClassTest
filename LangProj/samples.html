Arithmetic data can be either fixed point or floating, both using in decimal or binary as its base. An example of each of these combinations:

TEST: PROCEDURE OPTIONS(MAIN);

DECLARE XB FIXED BINARY(16);

DECLARE XD FIXED DECIMAL(8,2);

DECLARE FB FLOAT BINARY(8);

DECLARE FD FLOAT DECIMAL(5);

XB = 42;

XD = 123.45;

FB = 10110.01B;

FD = 123.45678;

PUT FILE (SCARDS) LIST(XB, XD, FB, FD);

END TEST;


which will print


42 123.45 2.22E+01 1.2345E+02



Following is the PL/I flavor of the "Hello World!" program. Comment statements are contained within /* */ - just like C++.

/* Hello World in PL1 */

Hello: /* each procedure block is given a name - in this case, Hello */

procedure options(main); /* declares this is the main part of the program */

put list('Hello World!'); /* prints "Hello World!" */

put skip; /* skips a line */

end Hello; /* tells the compiler which procedure block to end. This is likely a very useful feature for eliminating ambiguity within nested loops or conditional statements. */
